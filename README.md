# 华夏剧典

**以朝代为锚点，纵览两千年华语历史剧版图。**

从战国的纵横捭阖到民国的风云际会，华夏剧典将散落在各平台的历史剧集，以一条纵向时间线重新编织——让你在一屏之内，看见每个朝代最值得追的剧。

---

## 为什么需要华夏剧典

华语历史剧数量庞大，但缺少一个以「朝代」为坐标的全景索引。你可能遇到过这些问题：

- 想看唐朝背景的剧，搜索结果却混杂着现代穿越、古装甜宠
- 刚看完《大明王朝 1566》，想知道同朝代还有哪些高分剧
- 对某段历史感兴趣，却不知道有没有对应的影视作品

华夏剧典正是为此而生。它不推荐、不排行，只做一件事：**把历史剧放回它们所属的时间坐标里**。

## 谁会用到它

### 历史剧爱好者

按朝代浏览，一键筛选「严肃正剧」或「史事演义」，快速定位下一部想看的剧。每部剧都标注了豆瓣评分、集数和播放平台，省去反复搜索比价的麻烦。

### 历史爱好者

每部剧集标注了「历史锚点」（如"安史之乱前后"）和「核心张力」（如"忠诚与背叛的灰色地带"），帮你判断一部剧讲的是哪段历史、聚焦什么冲突——用影视作品为自己搭建一条非正式的历史认知路径。

### 内容创作者与影视研究者

覆盖从春秋战国到民国的完整朝代光谱，直观呈现各时期历史剧的分布密度和题材偏好——哪些朝代是热门 IP，哪些仍是创作蓝海，一目了然。

## 体验

深色博物馆主题，暖灰底色搭配金色时间线，每个朝代拥有独立配色。滚动时光束沿时间轴流淌，朝代标题 sticky 固定在视口左侧——浏览数十部剧，也不会迷失方位。

点击任意剧集，右侧滑出详情面板：海报、评分、历史背景、核心冲突、观看入口，一张卡片讲完一部剧的来龙去脉。
